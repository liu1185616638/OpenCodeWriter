# 小说创作方法论

## 一、大纲构建

1. **确定核心冲突**：每个好故事都需要一个核心矛盾，围绕它展开一切
2. **建立三幕结构**：开端（设定与触发）→ 发展（升级与转折）→ 结局（高潮与解决）
3. **设计人物弧线**：主角需要在故事中经历内在转变
4. **埋设伏笔**：在早期设置线索，让后期发展自然可信
5. **节奏把控**：紧张与缓和交替，避免全程高能或全程平淡

## 二、人物塑造

1. **动机优先**：每个重要角色都有清晰的内在动机
2. **性格多面**：避免单一标签，让角色拥有矛盾特质
3. **语言个性化**：通过对话风格区分角色
4. **成长弧线**：角色应在故事中有所变化
5. **关系网络**：角色间的关系推动情节

## 三、情节设计

1. **因果链**：每个事件都应有原因和后果
2. **意料之外，情理之中**：转折要惊喜但合理
3. **信息控制**：控制读者知道什么、角色知道什么
4. **递进升级**：冲突和赌注不断升高
5. **收束呼应**：结局要回扣开头的承诺

## 四、正文写作

1. **展示而非告知**：用行动和细节传达信息
2. **感官细节**：调动视觉、听觉、触觉等感官
3. **对话推进**：对话应推动情节或揭示人物
4. **场景切换**：控制节奏，适时跳过无聊部分
5. **结尾钩子**：每章结尾让读者想继续阅读
